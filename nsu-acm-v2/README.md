# NSU ACM Student Chapter Website v2.0

A modern, clean website for North South University ACM Student Chapter built with Next.js 16 and Tailwind CSS 4.

## Tech Stack

- **Next.js** 16.0.3 (Latest)
- **React** 19.2.0
- **TypeScript** 5
- **Tailwind CSS** 4 (Latest)

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
nsu-acm-v2/
├── app/
│   ├── layout.tsx      # Root layout
│   ├── page.tsx        # Homepage
│   └── globals.css     # Global styles
├── public/
│   └── acm-logo.png    # ACM logo
└── package.json
```

## Features

- Clean, minimal homepage
- Responsive design
- Modern gradient background
- Smooth animations
- TypeScript support

## Build

```bash
npm run build
npm start
```
